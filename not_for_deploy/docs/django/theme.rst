====================
Developing the theme
====================

Install deps::

    $ cd ievv_opensource/ievv_opensource_theme/static/ievv_opensource_theme/
    $ npm install
    $ bower install

Building the CSS and JS (from less and coffeescript)::

    $ cd ievv_opensource/ievv_opensource_theme/static/ievv_opensource_theme/
    $ grunt

To rebuild on file change, use::

    $ grunt watch
